## List Products

<api method="GET" path="/api/empresa/productos" />

Retrieves all products for the authenticated business.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from authentication
</ParamField>

### Response

<ResponseField name="[].id" type="integer">
  Product ID
</ResponseField>

<ResponseField name="[].nombre" type="string">
  Product name
</ResponseField>

<ResponseField name="[].descripcion" type="string">
  Product description
</ResponseField>

<ResponseField name="[].precio" type="float">
  Product price
</ResponseField>

<ResponseField name="[].stock" type="integer">
  Available stock quantity
</ResponseField>

<ResponseField name="[].empresa_id" type="integer">
  Business ID that owns this product
</ResponseField>

<ResponseField name="[].categoria_id" type="integer">
  Category ID
</ResponseField>

<ResponseField name="[].imagen" type="string">
  Relative path to product image
</ResponseField>

<ResponseField name="[].imagen_url" type="string">
  Full URL to product image
</ResponseField>

<ResponseField name="[].categoria" type="object">
  Category details
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Category ID
    </ResponseField>
    <ResponseField name="nombre" type="string">
      Category name
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="[].created_at" type="string">
  Creation timestamp
</ResponseField>

<ResponseField name="[].updated_at" type="string">
  Last update timestamp
</ResponseField>

### Example Request

```bash
curl -X GET https://api.beanquick.com/api/empresa/productos \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept: application/json"
```

### Example Response

```json

    "id": 1,

    "descripcion": "Café negro de 12 oz",
    "precio": 15.50,

    "empresa_id": 5,
    "categoria_id": 2,
    "imagen": "productos/cafe-americano.jpg",
    "imagen_url": "https://api.beanquick.com/storage/productos/cafe-americano.jpg",
    "categoria": {
      "id": 2,
      "nombre": "Bebidas Calientes"
    },
    "created_at": "2024-01-15T10:30:00.000000Z",
```

## Get Single Product

<api method="GET" path="/api/empresa/productos/{id}" />

Retrieves a specific product owned by the authenticated business.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from authentication
</ParamField>

### Path Parameters

<ParamField path="id" type="integer" required>
  Product ID
</ParamField>

### Response

Returns a single product object with the same structure as the list endpoint.

### Error Responses

<ResponseField name="message" type="string">
  Error message
</ResponseField>

- `404` - Product not found or doesn't belong to your business
- `404` - "No tienes una empresa vinculada." if user has no associated business

### Example Request

```bash
curl -X GET https://api.beanquick.com/api/empresa/productos/1 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept: application/json"
```

## Create Product

<api method="POST" path="/api/empresa/productos" />

Creates a new product for the authenticated business.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  multipart/form-data (required when uploading images)
</ParamField>

### Body Parameters

<ParamField body="nombre" type="string" required>
  Product name (max 255 characters)
</ParamField>

<ParamField body="precio" type="number" required>
  Product price (numeric value)
</ParamField>

<ParamField body="stock" type="integer" required>
  Initial stock quantity (minimum 0)
</ParamField>

<ParamField body="categoria_id" type="integer" required>
  Category ID (must exist in categorias table)
</ParamField>

<ParamField body="descripcion" type="string">
  Product description (optional)
</ParamField>

<ParamField body="imagen" type="file">
  Product image (max 2MB, image formats only)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message: "Producto creado con éxito"
</ResponseField>

<ResponseField name="producto" type="object">
  Created product object with same structure as GET response
</ResponseField>

### Example Request

```bash
curl -X POST https://api.beanquick.com/api/empresa/productos \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "nombre=Café Latte" \
  -F "precio=18.50" \
  -F "stock=50" \
  -F "categoria_id=2" \
  -F "descripcion=Espresso con leche vaporizada" \
  -F "imagen=@/path/to/latte.jpg"
```

### Example Response

```json
{
  "message": "Producto creado con éxito",
  "producto": {
    "id": 15,
    "nombre": "Café Latte",
    "descripcion": "Espresso con leche vaporizada",
    "precio": 18.50,
    "stock": 50,
    "empresa_id": 5,
    "categoria_id": 2,
    "imagen": "productos/xyz123.jpg",
    "imagen_url": "https://api.beanquick.com/storage/productos/xyz123.jpg",
    "created_at": "2024-03-05T10:30:00.000000Z",
    "updated_at": "2024-03-05T10:30:00.000000Z"
  }
}
```

## Update Product

<api method="PUT" path="/api/empresa/productos/{id}" />

Updates an existing product owned by the authenticated business.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from authentication
</ParamField>

<ParamField header="Content-Type" type="string">
  multipart/form-data when uploading new image, otherwise application/json
</ParamField>

### Path Parameters

<ParamField path="id" type="integer" required>
  Product ID to update
</ParamField>

### Body Parameters

<ParamField body="nombre" type="string" required>
  Product name (max 255 characters)
</ParamField>

<ParamField body="precio" type="number" required>
  Product price
</ParamField>

<ParamField body="stock" type="integer" required>
  Stock quantity
</ParamField>

<ParamField body="categoria_id" type="integer" required>
  Category ID (must exist)
</ParamField>

<ParamField body="descripcion" type="string">
  Product description
</ParamField>

<ParamField body="imagen" type="file">
  New product image (replaces existing one)
</ParamField>

### Response

<ResponseField name="message" type="string">
  "¡Actualizado!"
</ResponseField>

<ResponseField name="producto" type="object">
  Updated product object
</ResponseField>

### Error Responses

**404 Not Found** - Product not found or doesn't belong to your business
```json
{
  "message": "No encontrado"
}
```

### Example Request

```bash
curl -X PUT https://api.beanquick.com/api/empresa/productos/15 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Café Latte Grande",
    "precio": 20.00,
    "stock": 75,
    "categoria_id": 2,
    "descripcion": "Espresso doble con leche vaporizada - 16oz"
  }'
```

### Example Response

```json
{
  "message": "¡Actualizado!",
  "producto": {
    "id": 15,
    "nombre": "Café Latte Grande",
    "descripcion": "Espresso doble con leche vaporizada - 16oz",
    "precio": 20.00,
    "stock": 75,
    "empresa_id": 5,
    "categoria_id": 2,
    "imagen": "productos/xyz123.jpg",
    "imagen_url": "https://api.beanquick.com/storage/productos/xyz123.jpg",
    "created_at": "2024-03-05T10:30:00.000000Z",
    "updated_at": "2024-03-05T15:22:00.000000Z"
  }
}
```

## Delete Product

<api method="DELETE" path="/api/empresa/productos/{id}" />

Deletes a product and its associated image from storage.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from authentication
</ParamField>

### Path Parameters

<ParamField path="id" type="integer" required>
  Product ID to delete
</ParamField>

### Response

<ResponseField name="message" type="string">
  "Producto eliminado correctamente."
</ResponseField>

### Error Responses

**403 Forbidden** - Product doesn't belong to your business
```json
{
  "message": "No autorizado."
}
```

### Example Request

```bash
curl -X DELETE https://api.beanquick.com/api/empresa/productos/15 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept: application/json"
```

### Example Response

```json
{
  "message": "Producto eliminado correctamente."
}
```
