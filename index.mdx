  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with BeanQuick in minutes
  </p>

  <Steps>
    <Step title="Set up your environment">
      Clone the repository and install dependencies for both backend and frontend.

      ```bash
      git clone https://github.com/SeboskiHub/bean-quick.git
      cd bean-quick
      ```
    </Step>

    <Step title="Configure the backend">
      Set up your Laravel backend with database and environment variables.

      ```bash
      cd back-end
      composer install
      cp .env.example .env
      php artisan key:generate
      php artisan migrate
      ```
    </Step>

    <Step title="Launch the application">
      Start both the API server and the React frontend.

      ```bash
      # In back-end directory
      php artisan serve

      # In a new terminal, from front-end directory
      cd front-end
      npm install
      npm run dev
      ```
    </Step>
  </Steps>
</div>
